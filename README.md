# 전국 병원 지도

병원급 이상(상급종합·종합병원·병원·요양병원·정신병원·치과병원·한방병원) 4,229곳을 지도에서 찾아보는 정적 웹앱입니다.
`index.html` 한 장으로 동작하며 빌드 과정이 없습니다.

## 데이터 출처

- 건강보험심사평가원 「병원정보서비스」 (opendata.hira.or.kr, sno=11925)
- 소아청소년과 전문의 보유 여부는 같은 자료의 「의료기관별상세정보서비스_진료과목정보」를 매칭해 추가
- 좌표는 주소 기반 카카오 로컬 API 지오코딩으로 보완
- 진료과목·응급실 운영 등은 조사 시점 기준이며 방문 전 재확인이 필요합니다.

## 기술 스택

- Leaflet + OpenStreetMap 타일
- 빌드 없는 정적 HTML/CSS/JS, GitHub Pages(main 브랜치) 배포
